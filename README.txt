== Installation ==

1. Download the .zip plugin file
1. Extract it and put the `exposify` folder into your plugins directory
1. In your admin backend visit `Exposify` in the `Settings` section
1. Insert your Exposify credentials and a name and slug for your properties page
1. Choose a page template of your current theme (e.g. if you want to use a fullwidth template)
1. Enter a template for the overview of all properties and one for a single property
1. Insert your CSS code to style the templates from above
1. Hit `Save changes` and be happy ðŸŽ‰

Note: We hide the properties page in the pages overview, but you can still access it to add it to your menus.
